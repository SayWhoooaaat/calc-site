Neodymium production furnace

Biochar furnace

Small, simple Biochar furnace, manually operated. Converts wood to biochar through pyrolysis. Biochar is chemically stable 100% carbon, meaning that it can be stored permanently in the ground. This is certified CO2-capture that can reverse global warming if done in larger scale.
